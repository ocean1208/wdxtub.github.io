第四十二周 - 群山谱

发表于 2017-03-31 | 更新于 2017-08-03 | 分类于 Gossip | |
949 | 3

高山的至爱是流水，峭壁的恩师是雷电。山脉的慈母是大地，险峰的严父是苍天。

更新历史

  • 2017.03.31: 完成初稿

因为公司周五搬家，老板顺手就把周六也放了，虽然只是从六楼搬到十楼，但办公环境比原来好太多了，还领了一个站立办公桌，再也不用担心久坐了。忽如其来的四日假期终于给我一个回家休整的机会，一口气忙了一个月，能回家歇歇，真好。

重新开始上班一个月了，感觉交出了一份让自己满意的答卷。我的博客 wdxtub.com 终于在本周告别了长达四个月的 beta 阶段，正式迈入 1.0 版本，虽然不能说是焕然一新，但整体的思路已经确定，接下来就是继续好好经营了。

更令我高兴的是，经过两个多星期的设计开发，一个简单的数据挖掘平台已经开发完成，不但能够完成基本的数据相关工作，更是建立了一套完整的工作流，从数据标注到训练集测试集，从算法到模型再到验证。只需用鼠标在页面上点击，就可以完成大部分工作，避免了重复输入命令的低效与可能出现的误操作。工作流的规范化不但可以提高工作效率，更能够积累和重复利用经验，不然很容易像狗熊掰玉米一样，到头来啥都攒不下来。不过现在整个平台还非常脆弱，功能也比较有限，路漫漫而其修远兮，吾将上下而求索。

为了能让更多感兴趣的朋友参与进来，我打算在 Github 开源这套简易的系统，名字叫做 Patriots，为的是向《合金装备 IV 爱国者之枪》致敬。更有意思的是，做着做着我发现 Patriots 和 W.I.S.E. 项目中有很多重合之处，干脆一不做二不休，合体好了，这样我的精力可以更加集中，专注把整个平台搞好。

做出这个决定之后我是特别激动的，感觉终于找到了一个方式，把自己一直想做的事情和工作本身结合到了一起，在这个过程中考虑到方方面面，学着把事情做好。之所以要开源，也是希望能让有缘人看到，而且一直提醒自己这是一个谁都能看到的项目，做起来就更要精益求精，不能太随意了。即使是简单的系统，往深处挖掘，扔会有无穷无尽的细节。不沉下心踏踏实实把事情做好，不真正做用户需要的东西，是不可能产生价值的。

而我们生命的意义，不就是为了创造价值么？

最后说说生活，为了养的小草们能够每天晒晒太阳，我用别人装修剩下来的板子在窗边搭了一个小花台，每天上班出门前把小草们搬到花台上，晚上下班再收回屋子里。虽然只是非常微小的改变，但仍然增添了不少生活气息。还在坚持早睡早起、健身与过午不食（晚上吃水果），当然，偶尔也会放纵自己大吃一顿。

要做的事情和能做的事情都有很多，要跟着自己的内心走，不撞南墙不回头。

英雄天性若群山，大气磅礴日月间。各领风骚成自我，千变万化总巍然。天塌同做擎天柱，地陷鞠躬赴深渊。
