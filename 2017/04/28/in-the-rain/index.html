第四十六周 - 在雨中

发表于 2017-04-28 | 更新于 2017-04-28 | 分类于 Gossip | |
899 | 3

人生本是一段戏，有欢笑，也有哭泣。不知谁能，谁能躲得过去。

更新历史

  • 2017.04.28: 完成初稿

因为劳动节假期的临近，这一周似乎过得特别快。还记得清明节假期被代码支配的恐惧，忽然有几天可以休息一下，却还是一如既往地早起（早不早睡就不一定了），不过主要是因为住的地方在进行消防改造，一大早就开始叮叮当当，根本停不下来。

总体来说，这周大概是开始新工作之后博客产出最多的一周（居然转眼都入职了俩月了）。一是因为工作需要又开始读书，有了几篇读书笔记，二是因为博客访问量终于突破了一百万次。虽然在这个公众号动辄 10W+ 的年代，一百万看起来简直是个小数，不过对于我这种写作业余爱好者，攒了这么久有这个数字，也是很高兴的。

人嘛，在不断前的途中，就是不断需要各种各样的『意义』来鼓励自己的。无论目标是什么，能够完成，能够提前完成，都是令人开心的事情。接着世界读书日的名义重新恢复晨读之后，还是有很多收获的，还真应了那句话，大意是说几天不读书人就变傻。有一本说大数据的书，是 14 年写得，现在看来依然字字珠玑，很多陷阱和坑还是不断有人有公司前赴后继。我想，如果能够提前读到这本书，可能就能少走一点弯路吧。

说到写作，『我是范雨素』这篇文章给我的震动无疑是巨大的。的确，她的遣词造句说不上专业，但是字里行间流露出来的那种原生态和幽默感（加起来就是灵性）是我看一眼就能意识到差距的。有一句话是这么说的『论大部分人的努力程度，是根本轮不到拼天赋的』，但真的当一个人能够意识到天赋的差距的时候，大约是会绝望的，因为这很可能是努力再多都无法弥补的。不过真正热爱生活，有勇气的人并不会为此而改变自己，即使遥远的彼岸注定永远达不到，但是能多前进一点，也能多看几分风景，也就足够了。

在创业公司的好处在于总是得逼着自己去学各种各样的新东西，然后快速应用的实战中，这也意味着什么『xxx 是最好的语言/框架/算法』的思维定式是不存在的，我的目标就是先把成果搞出来，花里胡哨的可以后面再说，重要的是出活，而不管黑猫白猫，抓到老鼠就是好猫。

上周末和小伙伴一起去华南植物园团建了一波，满满的回忆杀，逗着小伙伴的宝宝，感觉见到啥都想放到嘴里的设定简直醉人，越看越萌。另外等了八个多月的 FPV 眼镜终于下个月要发货啦，这年头就是花钱买体验，有了好的体验，别说 2999，就是 3000 我也会毫不犹豫的！

你说人生艳丽我没有异议，你说人生忧郁我不言语。只有默默的承受这一切，承受数不尽的春来冬去。
