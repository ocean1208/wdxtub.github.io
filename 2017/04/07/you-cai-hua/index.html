第四十三周 - 油菜花

发表于 2017-04-07 | 更新于 2017-08-03 | 分类于 Gossip | |
1,044 | 3

一条大路呦，通呀通我家。我家住在呦，梁呀梁山下。山下土肥呦，地呀地五亩啊。五亩良田呦，种点啥。

更新历史

  • 2017.04.07: 完成初稿

万万没想到四天的清明假期居然过得这么快，一路堵车从广州回到深圳的我甚至有种天上一天地上一年的感觉。在家的日子，和老爸一起看球跟坐恒大牌山车，一起逛超市结果只买了十多罐酸奶，一起吃饭喝酒吹牛，都是特别美好的记忆。

每每累了倦了困惑了，回家，总是一剂特效药。在家的温暖中，我得以卸下所有的负担，全力以赴去努力和拼搏。能用自己的微薄之力让这个世界变得更好一些，其实我就心满意足了。青春大多迷茫，但只要不停下，总有那么一个时刻，会找到自己的方向。甚至自己都不一定发现，就像回到深圳后发现养的小草都发芽长新叶子了，这才联想起广州的嫩绿（注意发音，笑）。很多改变都是悄无声息的，我能做的就是在改变发生之前，一直走下去。

生活就是选择，无所谓好，无所谓坏，更多的是心安，只要很多年后想起当年的决定并不后悔，那就足够了。在面对巨大压力的时候，规律生活的重要性就更加凸现出来，规律才有节奏，有节奏才能心静，心静才能格物致知。没有夜生活，不喜欢吵吵闹闹，想要飞得更高，看更有趣的世界，我放弃了很多，但得到的也不少。

从 demo 到产品的距离，远比从前我认知的要长，真正去理解用户很难，真正能把握住重点的人很少，真正能每个时刻都做出正确决定的人不存在，这就是为什么做事情需要一个团队，这样才能在前进中找到动态平衡，不至于倾覆。所谓工作，就是不断的产出，不断的思考解决方案，不断解决问题，那些机械重复的工作终将被计算机取代，我一直是这么坚信的。

经过一年的磨练，自己的工程能力已经上了几个大的台阶，从产品到设计稿到前端到后端到算法到机器学习到存储再到测试，我基本上都有自己的方法论和套路，这样做起事情来就很舒服，不断尝试不断改进，最终效率会越来越高。撕逼总是能找到一万个理由的，好好说话则相反，需要大家都力往一处使，但是这并不简单。

小公司里很容易推动的东西，在大公司里很难，一是因为公司大了大家各有势力范围，求稳而不求变，二是因为真知灼见往往出现在一线，但是做事情流程太长，好点子早就过了半衰期。虽然不能马后炮的说自己之前制定的方案很科学合理（毕竟我最近看的企业白皮书很多成功实践和我的想法如出一辙），但其实也能说明一些问题。重点其实是有没有做事情的想法，列出一二三四和轻重缓急，然后想办法推动就是了，但可能是某个瞬间让自己意识到很多事情可能无能为力，也就罢了，人挪活嘛。

接下来还有很多有意义的事情等着我去做，我要好好锻炼身体，用最佳的状态去迎接各种挑战。已经意识到了 2017 是不平凡的一年，如果注定要因为一些人一些事不平凡（对我自己来说），那么为什么不能是我和我做的事情呢？

谁的欲望，谁的战场。我用长剑，劈断目光。谁会记得，我的模样。
