第六十三周 - 凡人歌

发表于 2017-08-26 | 更新于 2017-08-26 | 分类于 Gossip | |
1,292 | 4

一往无前虎山行，拨开云雾见光明，梦里花开牡丹亭，幻象成真歌舞升平。

当台风天鸽要擦过深圳的时候，所有人应该都没有意识到这会是和去年的台风妮妲一样可怕的怪物。还记得当时早早就挂上了台风红色预警，大家也都乐得浮生偷得半日闲。而天鸽的技能是暴怒，直接从蓝色预警旋转跳跃成红色。早起的朋友们不得不冒着大风大雨到公司然后接到放假通知但因为外面狂风暴雨哪也去不了，晚起的朋友们可以安心放下手机继续睡个回笼觉。对我来说问题倒也不大，因为无论是在家还是在公司，都是写写代码完善完善系统，其实并没有什么差别。

上周末和子妍一起去买鞋子修鞋子，顺带买了新碗新碟子新炖锅以及传说中的『阳光玫瑰葡萄』，不得不说这个葡萄啊简直亦可赛艇，丝毫没有酸味不说，还特别结实饱满无籽，要不然怎么价格是其他葡萄的五倍多呢（经过我们查阅资料，说国内现在也在逐渐引进要成为下一代的葡萄品种，真是太好啦）。炖锅顾名思义注意就是用来『炖』啦，比如胡萝卜骨头汤苦瓜排骨豆子汤以及酿豆腐等等，烧出来的味道就是跟之前买的钢锅不一样。还记得从前总是炖一大锅然后一连吃好几天，现在已经不会这样了，不同阶段有不同的饮食，一人食可能稍微随便些，但是二人食就得有些讲究了，生活中处处是学问，需要好好钻研积累，才能把日子过好。

时隔大半年终于又见到猜猜，那自然是少不了喝酒的，又想起当年猜猜、肥鱼、曲鳖我们四个在三藩喝酒的晚上（仅此一晚），虽然聊得内容只能依稀记得，但和老朋友一起把酒言欢的快乐，倒是深深印在脑海中了。人生短暂，还是要努力去创造不一样的记忆和经历，不然就太没意思啦，在这里祝猜猜的『返校』生活一切顺利，找时间一定去湾区聚。

这周和快递打了不少交道，先是顺丰的速度让我震惊，前一天晚上发出第二天早上就到了，冰块甚至都还没有融化，这种执行力我是真的服。另外就是在楼下的丰巢寄快递，小哥收到了取件短信但是来了却打不开柜门，和客服联系之后发现柜子里是有东西的，但是申通的系统没有对接上，可是奇怪的是小哥既然收到了短信，应该是开始了对应的流程的，这只能说明丰巢/申通的系统中可能在特殊情况下有 bug。不过这种涉及到不同公司不同的系统，其实很难保证同一流程中的每一步都不出错。换个角度来看，出错是正常的，只要错误是可以检测的，能够快速定位的，有明确输入和输出的，那么定时对依赖关系做检测，就可以保证系统的健壮。

我负责开发的深度学习平台已经准备接入线上和线下的流程的（小量和中量测试基本完成），估计接下来我还要继续折腾一段时间，所以也打算在周记中简单分享下感想。

  • 分布式系统的难点在于一致性、任务依赖
  • 一致性包括存储一致性和状态的一致性，其中存储的一致性现在已经有很多现成的方案，但状态的一致性除了轮训和占位之外，没有什么特别优雅的方法。因为对于状态来说，观测和实际使用之间的时间差导致了极端情况下的状态不稳，比如上一秒检测到 GPU 空闲，但可能这一秒派发任务的时候对应的节点就失联了（这类跨集群内外网的网络细节是非常让人头皮发麻的）
  • 任务依赖主要是一个状态机问题，这里的难点在于清晰的依赖定义与合理的失败重试机制

正式上线之后肯定还会有数不清的新问题和新需求，但是我觉得问题不大，毕竟遇到问题最重要的能力就是不慌，不慌才有后面的发挥。至于架构设计和编码能力都是在实践中慢慢提高的，只要心态不爆炸，总有运气和实力凑到一起的时候。

路上苦头不少，烈马长枪握好，你看我巍巍河山，中华屹立不倒。
