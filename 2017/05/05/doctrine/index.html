第四十七周 - 道

发表于 2017-05-05 | 更新于 2017-05-05 | 分类于 Gossip | |
963 | 3

道可道非常道，天道地道，人道剑道。黑道白道黄道赤道左道右道，有道无道人人说道他非常道。

更新历史

  • 2017.05.05: 完成初稿

五一的三天假期转眼过去，相比于疲劳的清明，我总算是有时间好好休息一下了。因为大巴超售车票没回成家，所以就在深圳把假期『晃悠』过去了。去小河边骑了单车飞了飞机，去新开的商场逛了书店，去宜家参考了一下家装的套路，于是假期结束后就开始忙碌了起来。

在整租的房子里住了三个月之后，结合平时生活的一些痛点，我设计了一套提高空间利用率的方案。从网上把各种架子买回来，拼拼装装挪挪放放，整个屋子焕然一新。秘诀很简单，主要是用各种架子提高了空间利用率，然后把屋子进行不同的功能分区。今天收到了厨房的架子，这样最后一个步骤就完成了。现在整个家里井井有条，看着就舒服。

工作中，因为很多工作都是探索性的，所以难免遇到各种困难。创业公司嘛，最重要就是快，就是要出活，一山放过一山拦是肯定的。所谓工程师，就是不断遇到问题，不断尝试，最终解决问题的人。

写代码也有七八年了，经手的项目也有十来个了，做工程的艺术，其实就是对二八原则的理解、运用和升华。很多时候我们不是要最好的，是要性价比最高的。虽然我们总是自嘲自己的工作是『搬砖』，但就写写代码这个事儿来说，绝不是表面看上去的那么简单。就像书籍的作者，如果说书中的文字是作者的内心，那么我们写的代码，很多时候就是我们看待世界和改造世界的方式和想法。工作中遇到的各种问题，是把问题当问题，还是把问题当修行，还是把问题当探索，还是把问题当艺术都可以，就是不要把问题当麻烦。因为如果一开始就当成是麻烦，就错过了一个明心见性格物致知的机会，所谓内心，就是在不断磨砺中愈发闪亮的。

二八原则搭配上不同的领域其实是有不同说法的，很多人知道说二八原则可以理解为把事情做到 80 分大概只需要 20% 的力。那么这个领域的平均分是多少，是有极大影响的。如果大家的平均水平是 60 分，那么 80 分也不要高兴得太早，因为大家连 20% 的力还没用；如果大家的平均水平是 90 分，那么也不要太难过，因为这说明竞争程度已经很高，花大力气投入可能还不如直接买现成的。如果大家的平均水平是 80 分，那么恭喜，当你达到 80 分的时候，游戏才真正开始，接下来就是拼努力和天赋的时候了，所谓坚持就是胜利说得是这个阶段。

所以当我们想要到一个领域里试水的时候，弄清楚到底是多少分，很重要，甚至比 80% 的努力都重要，也就是所谓的选择比努力更重要，最重要的是选择了之后一直努力。

邪门歪道时来运到旁门左道生财有道，摆我一道不能人道汪洋大盗横行霸道鬼哭神号豺狼当道，
我自求我道。
